## üéì Understanding the Model

The Credit Card Fraud Detection application uses advanced machine learning techniques to analyze transaction data. Here are a few concepts explained:

- **Random Forest**: A type of model that uses multiple decision trees to improve accuracy.
- **XGBoost**: Another model known for its speed and efficiency in prediction tasks.
- **SMOTE**: A technique to balance the dataset by creating synthetic examples of the minority class (fraudulent transactions).

These techniques work together to improve the detection of fraud while minimizing incorrect flags on valid transactions.
